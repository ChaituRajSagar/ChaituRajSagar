### 🏛️ GovGenie — AI RFP Response Generator
Retrieves past proposal language from ChromaDB semantically, drafts government bid responses in company voice. Conversational chat interface with LangChain memory for follow-up clause questions.
Reduced bid prep: **5+ days → under 24 hours**

### 📄 ReferenceFiller — DOCX Automation
Upload resume → LLM extracts structured data → fills DOCX template via semantic field mapping. Session-based UUID stores, chunk-based text splitting, rotating log handlers.
Replaced: **45-min manual task → 2-min automated workflow**
